IoT Health Kit Monitoring System


Introduction

Technology plays the major role in healthcare not only for sensory devices but also in communication, recording and display device. It is very important to monitor various medical parameters and post operational days. Hence the latest trend in Healthcare communication method using IOT is adapted. Internet of things serves as a catalyst for the healthcare and plays prominent role in wide range of healthcare applications.

Routine medical checkup is a preventive measure to avoid illness and improve health quality. However, not everyone has the time to do so and some are constrained by the distance between home and the hospital. This condition causes a demand of a system capable of monitoring oneâ€™s health status that is not only portable, but also low in price so that it is accessible to people of all economic level.

Methodology

This project uses Arduino Uno as microcontroller to interface different health sensors. This healthcare kit comprises of a temperature sensor, heartbeat sensor and ECG (Electrocardiogram) sensor. The arduino picks up the sensor data and sends to the network through GSM/GPRS module and hence provides real time monitoring of health care parameters for doctors. The reading values of different sensors are displayed by a LCD display. Also, these readings are uploaded to a webpage in real-time which can be viewed by authenticated personnel such as doctors or patients themselves. The data can be accessed anytime by doctors or patients. At emergency condition an alert SMS message and email is sent to the doctor by GSM/GPRS module connected to the controller. This prototype enables users to improve health related risks and reduce healthcare costs by collecting, recording, analyzing and sharing large data streams in real time and efficiently. This project helps to reduce the headache of patient to visit to doctor every time he need to check his heart beat rate, temperature etc.

  • Temperature Sensor

Applications

  • It is used to display patient parameter in the various hospitals.
  • It can be used in remote areas where there is lack of hospitals and doctors.
  • It can be used in old age homes to get to monitor the various parameters of a sick person in old age homes.
  • It is very important for regular visiting patient whose health status is monitored regularly.

FUTURE DEVELOPMENT

In addition to the system can also provide more than one numbers so that more than one user can receive emergency message. According to availability of sensors or development in biomedical trend more parameter can be sense and monitor which will drastically improve the efficiency of the wireless monitoring system in biomedical field. Furthermore- this project can be enhanced by using different parameters such as:

  • Retinal size, weight and age can be included to control the parameters in the future.
  • This system can be developed by using advanced technologies like GPS. In future we can send this data to a remote internet and we can add the module of voice alarm system to indicate parameters crossing the threshold value.
  • Patient voice recognition system: IC HM2007 can be used to recognition the voice samples of the patients for the better security purpose.
  • A camera can be fitted into the system so as to enable the base station to get a real time view of the battlefield.
